### 定期レポート（5機能）
- 日次・週次・月次・年次レポート
- テスト実行レポート

### 分析レポート（5機能）
- ライセンス・使用状況・パフォーマンス分析
- セキュリティ・権限監査

### サービス管理（16機能）
- **Entra ID管理**（4機能）: ユーザー・MFA・条件付きアクセス・ログ
- **Exchange Online管理**（4機能）: メールボックス・フロー・スパム対策・配信
- **Teams管理**（4機能）: 使用状況・設定・会議品質・アプリ分析
- **OneDrive管理**（4機能）: ストレージ・共有・同期・外部共有

## 🔧 技術仕様

- **PowerShell**: 7.5.1推奨・5.1最小サポート
- **Python**: 3.11以上（PyQt6対応）
- **認証**: Microsoft Graph・証明書ベース
- **出力**: CSV（UTF8BOM）・HTML（レスポンシブ）・PDF
- **ログ**: 構造化ログ・監査証跡対応
- **tmux**: 3.4以上推奨
